## Prerequisites

- [Python] 3

- [MkDocs] 1.1+
  with the [Macros] and [Mermaid] plugins and the [Material] theme

## Development

```console
$ git clone https://github.com/aurora-is-near/doc.aurora.dev.git

$ cd doc.aurora.dev

$ pip3 install -r requirements.txt

$ mkdocs serve
```
